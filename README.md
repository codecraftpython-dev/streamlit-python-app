# streamlit-python-app
# 🚀 Streamlit Apps Collection

This repository contains two simple and interactive web applications built using **Streamlit (Python)**.

---

## 📌 Applications Included

### 📝 User Form

* Collects basic user details (name, age, email, gender)
* Includes validation for required fields
* Displays submitted data

### 🎮 Tic Tac Toe Game

* Interactive Tic Tac Toe game
* Built using HTML, CSS, and JavaScript
* Integrated into Streamlit

---

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone <your-repo-link>
cd <repo-folder>
```

2. Install dependencies:

```
pip install -r requirements.txt
```

---

## ▶️ Run the Applications

### Run User Form:

```
streamlit run form_app.py
```

### Run Tic Tac Toe Game:

```
streamlit run tic_tac_toe.py
```
